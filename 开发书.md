# 开发书

## 项目
- 名称：文文Tome
- 当前版本：1.8.0+14
- 当前目标：把导入、阅读、TTS、翻译模型、API 配置、日志和打包链路全部做成可交付状态。

## 当前架构
- `lib/app/`：路由与主题
- `lib/features/library/`：书库、导入、元数据
- `lib/features/reader/`：阅读器、TTS、本地 Piper、正文探测
- `lib/features/translation/`：本地翻译模型、API 调用、配置管理
- `lib/features/settings/`：设置页、环境检测、日志页、关于页
- `lib/features/logging/`：运行日志服务与动作封装

## 本轮重点修复
- Piper 中文标准声线改为内置
- 翻译模型下载初始化竞态修复
- API 配置页返回逻辑修复
- EPUB 正文提取与稳定展示
- 运行日志固定落盘并支持导出/分享/清空
- 中文标题、默认配置、安装相关乱码清理
