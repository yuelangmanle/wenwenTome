# 2026-03-10 项目开发进度书（交接版）

## 1. 项目目标（当前口径）
- 目标：把 `wenwen_tome` 做成“本地书阅读 + 网文搜书/入库/缓存 + 稳定 TTS”的可用版本。
- 关键要求：移动端可稳定使用，不卡死、不闪退、加入书架后可读。

## 2. 版本演进摘要
- `2.0.0`：网文模块重构为规则驱动仓储，初步打通搜书与网页识别链路。
- `2.1.0`：移动端启动与下载恢复修复，TTS/翻译下载链路稳态化。
- `2.2.0`：分页阅读、翻页动画、自定义字体、主题等能力补齐。
- `2.3.0`：大书源导入与兼容增强，新增搜书大师对照报告与全模块借鉴报告。

## 3. 模块进度总览（截至 2026-03-10）
- 书架与导入：`85%`
- 书架与导入已完成：多格式导入主路径、元数据补全、基础兼容。
- 书架与导入未完成：部分格式深度解析体验仍不稳定（超大 TXT、复杂 EPUB、复杂 PDF）。
- 阅读器（文本/分页/菜单）：`90%`
- 阅读器已完成：分页与滚动双模式、主流字体与动画、目录与书签链路、菜单显隐与误触收敛。
- 阅读器未完成：进入更大样本长时回归与发布闭环（并入 `QAT/REL`）。
- 网文搜书与网页搜索：`96%`
- 网文已完成：书源导入、搜书、网页识别、加入书架、入库回退与主链路可读闭环；评分/失败报告/查询扩展/并发调节等质量工程化能力。
- 网文未完成：更大样本书源回归与少量站点规则补齐（非阻塞）。
- 缓存下载：`90%`
- 缓存已完成：书级/章节级任务化、任务落库与后台恢复、失败原因分类、空间配额与清理策略、缓存管理页（暂停/继续/清除/统计/配置）。
- 缓存未完成：进入更大样本弱网回归与发布闭环（并入 `QAT/REL`）。
- TTS：`86%`
- TTS 已完成：多引擎入口、模型管理、分段队列、降级与中断恢复、会话控制器。
- TTS 未完成：进入质量回归与发布闭环（`QAT-001~007`）。
- 翻译模块：`40%`
- 翻译现状：本地翻译链路维护成本与复杂度偏高。
- 翻译建议：收敛到统一 AI API 路径，逐步下线本地大模型入口（`TRN-001~004`）。

## 4. 今日关键进展（本次会话已完成）
- 版本：`2.6.9+35`（质量闭环与发布检查）。
- WDL-001~008：缓存下载任务服务化（书级→章节级队列、落库、恢复、失败分类、覆盖与清理策略、缓存管理页）。
- FRM-007：PDF 目录页码映射归一化，支持按目录跳页，阅读进度/书签按页持久化。

- RDR-004（性能）：目录弹层改为懒加载列表，分段分页跳转请求可在分页完成后生效，避免大目录卡顿。
- 继承 `2.6.6+32`：搜书质量工程化（WNS-005~010）与导入链路规范化（FRM-001/002）。
- WNS-005：搜书结果质量评分与去噪（直连源优先、疑似搜索/索引页降权）。
- WNS-006：失败分级提示与可操作建议（降并发、启用扩展、缩小书源范围）。
- WNS-007：书源导入报告增强（imported/updated/skipped/failed + `updatedCount`）。
- WNS-008：书源导出仅包含自定义书源，支持“导出→再导入”闭环校验。
- WNS-009：识别页入库候选 URL 优先目录/TOC，提升入库后拉目录命中率。
- WNS-010：搜索关键词自动扩展（可关闭），提升弱关键词命中率。
- FRM-001/002：导入格式白名单与导入来源统一（`file://`/`content://` 规范化，提示补齐 `CBR`）。
- QAT/REL：回归集脚本、关键路径结构化运行日志、超时/取消策略收敛、发布前检查脚本与回滚方案文档补齐。
- 可复现测试步骤：
1. 搜书页切换“查询扩展/并发度”，用弱关键词搜索并对比命中率与超时率。
2. 书源管理页导出自定义书包后立刻导入，检查字段不丢失，导入报告可复制并能定位失败条目。
