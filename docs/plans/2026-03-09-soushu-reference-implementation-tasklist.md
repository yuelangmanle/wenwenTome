## 完成定义（DoD）
- 功能 DoD：真机可操作、错误可解释、失败可恢复。
- 性能 DoD：无明显卡死，关键操作可在可接受时延内完成。
- 质量 DoD：新增或改动链路有自动化回归，且日志可追踪。
- 文档 DoD：每个模块有“现状/已知限制/下一步”记录，便于交接。
