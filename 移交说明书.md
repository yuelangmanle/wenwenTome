# 移交说明书

## 交付内容
- Android 安装包：`releases/2.0.1/wenwen_tome_android_2.0.1.apk`
- Windows 安装包：`releases/2.0.1/wenwen_tome_windows_2.0.1_setup.exe`
- 更新日志：`CHANGELOG.md`
- 版本说明：`releases/2.0.1/release_notes.md`
- 运行日志目录：开发环境默认位于项目根目录 `运行日志/`

## 本版重点
- Android 启动链改为非阻塞启动，初始化异常时可降级进入主界面
- 书架和网文页在存储或仓储初始化失败时不再无限转圈，支持提示与重试
- Windows 安装器中文资源编码修复，安装界面中文可正常显示
- Windows PDF 阅读链路修复，中文路径与字节流打开更稳定
- 网文搜索与网页模式入库链路修复，支持更可靠的目录页归一化

## 验收建议
1. Android 冷启动应用，确认不会卡在“正在准备启动环境”
2. 打开书架和网文页，确认初始化异常时会显示可重试反馈
3. Windows 安装版直接打开一个 PDF，确认能正常进入阅读
4. 导入一个 Legado 3.0 书源或粘贴 `legado://` 链接，确认能识别导入
